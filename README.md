# README.md

## Сантехнарь (Santehnar)

---

### Описание

Сантехнарь - это веб-сайт сети магазинов санитарной техники. Проект написан студентом Бердниковым А.С. группы ДИС-233/21.Б на PHP в фреймворке Laravel.

---

### Планируемый функционал

- обший шаблон страниц: шапка, подвал, главная обёртка
- регистрация и аутентификация пользователей по почте и паролю
- личный кабинет, редактирование почты и пароля
- восстановление пароля через почту
- возможность регистрации как администратора
- каталог товаров и фильтрация по цене, названию, типу
- страница одного товара с деталями: изображением, описанием, хакартеристиками, отзывами
- отзывы пользователей к товарам и рейтинг
- корзина: добавлять, убирать, выбирать кол-во выбранного товара, статус товаров
- просмотр истории заказов в личном кабинете
- админ-панель: управление товарами и их статусами, категориями товаров, отзывами, пользователями
- адаптив
- seo

### Как установить и запустить

1. Клонировать репозиторий и зайти в него:
    ```bash
    git clone https://github.com/tapotheslipper/Santehnar.git <путь к вашей директории>
    cd <путь к вашей директории>
    ```
2. Установить зависимости:
    ```bash
    composer install
    ```
3. Скопировать .env.example в .env и заполнить необходимыми значениями.
4. Сгенерировать ключ приложения:
    ```bash
    php artisan key:generate
    ```
5. Мигрировать базу данных:
    ```bash
    php artisan migrate
    ```
6. Запустить сервер:
    ```bash
    php artisan serve <свой порт, необязательно>
    ```
